Contributions of all kinds are welcome: bug reports, feature ideas, documentation improvements, and code.

This project aims to be **stable, maintainable, and future-proof**.  
Please read the guidelines below to ensure smooth collaboration.

---

## 1. Ways to contribute

You can contribute by:
- Reporting bugs
- Suggesting new features or improvements
- Improving documentation
- Submitting pull requests
- Reviewing pull requests and discussions

If you are unsure where to start, open an issue and ask.

---

## 2. Before you start

- Search existing **issues** and **pull requests** before creating new ones
- Discuss larger changes in an issue first
- Keep changes focused and small where possible

---

## 3. Development setup
